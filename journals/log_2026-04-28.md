## Mechanistic story (final)

The original C_v3 negative result was a hyperparam miscalibration, not
a data problem:

1. **Top-4M PPO-RNN has wider RTG spread** (14.6–28.4) than the old
   top-2M (12.2–17.1). At default β=10, AWR's `exp(adv/β)` weights are
   *flatter* across this wider range — the highest-RTG demos get
   weighted similarly to mid-RTG ones.
2. With wider data and flatter weights, AWR effectively becomes more
   BC-like. Average behavior in PPO-RNN data is shorter-lived (mean
   ~317 steps) → the policy mimicked that.
3. **Higher β** (30, 50) restores AWR's focus on the highest-RTG demos
   (longest+deepest runs), and **adding the long-survival PSF v2 data**
   provides those demos directly.

So the data was always better; we just needed β to scale with the RTG
spread. **Higher β + more diverse data = the win.**

This is a useful general lesson for AWR on heterogeneous datasets:
don't keep β fixed when the RTG distribution widens.

**Optimizer collapsed after iter 2.** Iters 3-8 are all near -0.9
(immediate death — the proposer almost certainly broke the
`ACTION:` output format expectation, so the parser fell back to NOOP
every step and the player died on environmental decay). The collapse
is masked by the next iter's seed coming from the previous iter's
*output* — once the broken prompt is in the loop, the proposer is
working off a broken template and produces broken refinements.

**Bug**: my outer-loop design seeds each iter from `iter_{N-1}/prompt.txt`,
NOT from `best_prompt.txt`. A single bad proposal poisons the rest of
the run.

**Fix** (one-line change): seed iter N from the best-so-far prompt
rather than previous iter's output. With this, a bad proposal just
gets discarded and we try again from the best.

Best prompt this run = iter 1 at 8.10 raw return (+0.40 over seed
7.70). Saved at `best_prompt.txt`.

(Earlier note: an even earlier prompt_iter run had iter 0 = -0.90
because I'd seeded with the FUTURE-PREDICTION template instead of
the ACTION-SELECTION one. That run was cancelled; this v2 run has
the correct seed.)
